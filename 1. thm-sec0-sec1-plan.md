# Foundations Plan — Pre Security (SEC0) → Cyber Security 101 (SEC1)

My commitment: **one hour a day, one room per session.** Speed is a bonus, not the goal.
The only rule that survives a fast pace: **one journal entry per session, no matter what.**
Don't break the streak — a 20-minute day still counts; a zero day resets the fire.

> The plan: build a real foundation in order, no gaps. SEC0 first (7 modules, 31 labs, ~19h),
> then SEC1 (14 modules, 56 labs, ~46h). Two certificates. Then I'm ready for Jr Pen Tester (PT1).
> Checkboxes render as clickable on GitHub. To tick one in a committed file, edit it (✏️) and
> change `- [ ]` to `- [x]` — each edit is a commit = a green square.

---

## Free progress note
Some rooms appear in more than one path. Once a room is done, it's done **everywhere** —
THM marks it complete across every path. In particular, **Offensive Security Intro**,
**Defensive Security Intro**, and **Search Skills** show up again in SEC1 Section 1, so if I've
already done them, they're auto-ticked there. Free squares.

## How I journal each room
File named `YYYY-MM-DD-room-name.md` with:
- What the room covered (2–3 sentences, my own words)
- One tool or technique I learned
- One thing that tripped me up and how I got past it

Three sentences beats a blank page every time.
